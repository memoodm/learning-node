# 1. INSTALL PACKAGES

### install express

```
npm install express
```

### install project updater listening for changes in files

```
npm install nodemon -g --save
```

### install mongo database connection

```
npm install --save mongoose
```

# 2. INSTALL DATABASE

### Mongo
Install mongo db

### Display mongodb data

